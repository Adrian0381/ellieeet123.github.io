Proxy


A fancy little proxy service. This can pass any school filter, such as Securly or a firewall.


This requires a password to launch, and the password changes relatively frequently. The reason for this is to prevent the server from becoming overloaded, and to prevent it from being blocked. (If it is blocked, I can make a new one relatively easily but I'd rather not do that.) To obtain a password, either you will have to know someone IRL who can give it to you, or you can add 20 games to the site and get access. (See 'Help Build the site!' for more info)
